# 方块世界 3D 网页游戏

一个基于 WebGL 的第一人称方块世界游戏。游戏主体位于 `index.html`，通过 Next/Vite 应用壳运行，也可以使用静态服务器直接预览。

## 开发

```bash
npm install
npm run dev
```

构建与检查：

```bash
npm run build
npm test
npm run lint
```

## 操作

- `W A S D`：移动
- 鼠标：控制视角
- `空格`：跳跃
- `Shift`：疾跑
- 鼠标左键：挖掘方块
- 鼠标右键：放置方块
- 数字键：切换快捷栏
- 鼠标滚轮：切换快捷栏
- `Esc`：暂停并释放鼠标
